# 嘉義縣立水上國民中學校友會網站｜ssjh-alumni-next

這是新版「水上國中校友會」GitHub Pages 靜態網站草案。

## 網站定位

- 連結歷屆校友
- 保存水中校史與地方記憶
- 記錄校友會活動
- 支持母校教育、體育、藝文與獎助學金

## 檔案結構

```text
index.html
about.html
history.html
activities.html
scholarship.html
alumni-stories.html
join.html
contact.html
sources.html
assets/css/styles.css
assets/js/main.js
assets/img/favicon.svg
.nojekyll
robots.txt
sitemap.xml
```

## GitHub Pages 設定

1. 建立 repo：`ssjh-alumni-next`
2. 上傳本專案全部檔案到 repo 根目錄
3. 到 Settings → Pages
4. Source 選 `Deploy from a branch`
5. Branch 選 `main`，資料夾選 `/root`
6. 網址預期為：`https://elvismtlee.github.io/ssjh-alumni-next/`
